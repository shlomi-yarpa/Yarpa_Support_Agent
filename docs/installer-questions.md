# מידע שאנחנו צריכים מהטכנאים – Yarpa Support Agent

שלום,
אנחנו בונים כלי שאוסף אוטומטית מידע טכני ממחשב הלקוח (גרסת Windows, חומרה, מדפסות,
מסופי סליקה, SQL, גרסת Yarpa ועוד) כדי שצוות התמיכה יראה תמונת מצב מלאה עוד לפני
שמתחילים לטפל בתקלה.

כדי להשלים את הכלי, אנחנו צריכים ממכם כמה פרטים מהשטח. אפשר לענות בטקסט פשוט
ולצרף צילומי מסך היכן שמבקשים. תודה!

---

## 1. גרסת תוכנת Yarpa (הכי חשוב)

אנחנו רוצים שהכלי יזהה לבד שגרסת Yarpa מותקנת ומהי הגרסה.

1. **נתיב ההתקנה** – מה התיקייה המלאה שבה מותקנת התוכנה?
   דוגמה: `C:\Program Files (x86)\Yarpa\...` או `C:\Yarpa`

2. **קובץ ההרצה הראשי** – מה שם קובץ ה-EXE הראשי? (דוגמה: `Yarpa.exe`)
   בבקשה גם: קליק ימני על הקובץ → Properties → לשונית **Details** →
   צלמו את השדות **File version** ו-**Product version** ושלחו צילום מסך.

3. **רישום ב-Registry** – האם יש מפתח של Yarpa ברג'יסטרי?
   (לבדוק תחת `HKEY_LOCAL_MACHINE\SOFTWARE\Yarpa` או
   `HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Yarpa`)
   אם כן – מה שם המפתח ואיפה מופיע מספר הגרסה?

4. **"הוספה או הסרה של תוכניות"** – האם Yarpa מופיעה שם?
   אם כן – מה **השם המדויק** כפי שהוא מוצג? (דוגמה: "Yarpa ERP")

5. **קובץ גרסה** – האם יש בתיקיית ההתקנה קובץ שמכיל את מספר הגרסה?
   (דוגמה: `version.txt`, `config.ini`) אם כן – שם הקובץ והשורה הרלוונטית.

---

## 2. שירותי Yarpa (Windows Services)

1. אילו **Windows Services** מתקינה Yarpa? (שם השירות + Display name)
2. אילו מהם **קריטיים** (חייבים לרוץ כדי שהתוכנה תעבוד)?

---

## 3. גרסה מינימלית נתמכת

מהי **הגרסה המינימלית** של Yarpa שנחשבת "עדכנית"?
(מתחתיה נרצה להתריע "גרסה ישנה". דוגמה: 8.4.0)

---

## 4. מסופי סליקה

אנחנו רוצים שהכלי יזהה איזה מסוף סליקה מחובר למחשב.

1. אילו **יצרנים** נפוצים אצל הלקוחות? (Ingenico / Verifone / PAX / Castles / אחר)
2. אילו **דגמים** ספציפיים? (דוגמה: Verifone VX520, PAX A920)
3. איך הם **מחוברים**? USB רגיל / USB-to-Serial (COM) / רשת (IP)
4. **החשוב ביותר – מזהה החומרה (VID/PID):**
   על מחשב עם מסוף מחובר:
   - לפתוח **Device Manager** (מנהל ההתקנים)
   - למצוא את המסוף ברשימה → קליק ימני → **Properties**
   - לשונית **Details** → בתפריט לבחור **"Hardware Ids"**
   - לצלם מסך. זה נראה כך: `USB\VID_11CA&PID_0300`

   בבקשה שלחו את זה **לכל דגם מסוף** שאתם מתקינים, ואם אפשר גם
   באיזה **COM Port** המסוף יושב.

---

## 5. סורקים / קוראי ברקוד (אם רלוונטי)

1. אילו **דגמי קוראי ברקוד / סורקים** נפוצים? ואיך הם מחוברים
   (USB-HID / USB-Serial / COM)?
2. אם יש דגם חשוב לזיהוי – שלחו גם עבורו **Hardware Ids** מ-Device Manager
   (אותה שיטה כמו בסעיף 4).

---

## 6. SQL Server

1. באיזו **מהדורת SQL** אתם משתמשים אצל הלקוחות? (Express / Standard / אחר)
2. מה **שם ה-Instance** הנפוץ? (דוגמה: `SQLEXPRESS`, `MSSQLSERVER`, או שם מותאם)

---

## מה הכי דחוף

אם אין זמן לענות על הכל – שני הדברים הכי חשובים הם:
- **סעיף 1** (נתיב ההתקנה + צילום ה-File version)
- **סעיף 4** (ה-Hardware Ids של מסופי הסליקה)

תודה רבה על העזרה!
